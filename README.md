# My Personal Webpage

Hey there! In one of my assignments for the course Introduction to Software Systems, I was tasked with learning how to create my own webpage using HTML, CSS and vanilla JavaScript. It was supposed to contain 4 pages - a home page, an "about me" page, a projects page (containing information on some of the projects I've done) and a contact page (just tasked with front-end verification of input, no backend required). We were also tasked with implementing animations (like fade-ups, drop-downs, expandables and the like) and other cool features (like implementing a dark mode theme option, presenting a timeline of some of the important milestones in my life etc). Though it was very frustrating at times, the overall project was quite fun to implement, and especially considering that this is something I could customise at any time and deploy as my personal website, I'd say that I thoroughly enjoyed this project and learnt a lot from it.

in this repo I've included all the HTML, CSS and JS files I've written to create my website. I've also provided screenshots for the Lighthouse checks (with respect to Accessibility and Best Practices, for both dark + light mode) and the HTML w3 Validator checks for each of the HTML files. The link to my webpage is at http://web.iiit.ac.in/~shaun.thomas, feel free to check it out if you're interested! 

Have fun! :D
